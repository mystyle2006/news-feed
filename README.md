## Description

학교 소식을 전달하고 받아보는 '학교소식 뉴스피드' 를 위한 백엔드 구현

## Spec

- nestjs, graphql, typescript
- mysql
- class-validator, class-transformer, ts-jenum, dayjs

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
